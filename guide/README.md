# Secyud Tavern 使用指南

## 快速入门

1. **配置模型** → 进入 LLM API 标签，创建 API 配置（填入 Key、选择模型）
2. **获取预设** → 导入预设包或从预设列表中选择
3. **新建故事** → 选择模型 + 加载预设，输入开场白
4. **开始对话** → 点击故事旁的跳转按钮（↘），进入游玩界面
