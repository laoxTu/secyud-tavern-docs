# 正则 (Regex)

文本查找替换规则。在消息发送前和展示后对文本做正则替换。

![正则列表](../../images/regex-list.jpeg)

## 条目字段

| 字段 | 说明 |
|---|---|
| **Code** | 唯一标识符 |
| **Name** | 显示名称 |
| **Pattern** | 正则表达式（JavaScript 语法，不带前后 `/`） |
| **Replacement** | 替换文本（支持 `$1`、`$2` 等捕获组引用） |
| **Target** | 生效目标 |
| **Disabled** | 是否禁用 |

## Target 取值

| 值 | 说明 | 时机 |
|---|---|---|
| `input` | 用户发送前 | 用户输入 → 正则替换 → 发送给 AI |
| `output` | AI 回复后 | AI 回复 → 正则替换 → 展示给用户 |
| `both` | 双向生效 | 输入和输出都应用 |

## 使用示例

| Pattern | Replacement | Target | 效果 |
|---|---|---|---|
| `<variable_changes>[\\s\\S]*?</variable_changes>` | （空） | `output` | 剔除 AI 输出中未正确闭合的变量标签 |
| `\*{3,}` | `**` | `both` | 统一多余星号 |
| `\\[系统提示\\]` | （空） | `input` | 过滤用户输入中的系统标签 |

## 注意事项

- Pattern 直接写正则表达式内容，**不要**包裹 `/pattern/flags` 格式
- 需要 flags 时在表达式内使用 `(?flags)` 语法，或直接依赖默认行为（全局匹配）
- 多条正则按条目列表顺序依次执行
