# Linux Installation Guide

## Manual Git Installation

For MacOS / Linux, all operations will be done in the Terminal.

1. Install git and NodeJS (the specific method will vary depending on your operating system)
2. Clone the repository
    * `git clone https://github.com/laoxTu/secyud-tavern`
3. Navigate to the installation folder with `cd secyud-tavern`
4. Run the start.sh script using one of the following commands:
    * `./start.sh`
    * `bash start.sh`
5. The server will then start, and you can access it from your browser at `http://127.0.0.1:12804`

### Updating

1. Navigate to the installation folder
2. Pull the latest code with the following command:
   * `git pull`
3. Run the start.sh script using one of the following commands:
   * `./start.sh`
   * `bash start.sh`
