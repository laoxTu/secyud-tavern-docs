# User Guide

## [Models](model/readme.md)

Models provide LLM configuration. Here you can configure your API type, options, and keys for accessing LLMs.

## [Presets](preset/readme.md)

Presets are the core feature of Secyud Tavern, providing all interaction configuration. Here you can build your own interaction stage.

![Presets](../images/preset.png)

## [Stories](story/readme.md)

Stories are the entry point for save data. Each story is a save file, where you can play and browse the image gallery.
![Stories](../images/story.png)

## [ComfyUI](comfyui/readme.md)

ComfyUI provides image generation capabilities within stories.
![ComfyUI](../images/comfyui_model.png)
