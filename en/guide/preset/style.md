# Styles User Guide

![Styles](../../images/preset_style.png)

## Field Descriptions

* Priority: Determines the order in which styles are injected.
* Type: The type of style, determining how the style is injected.
    * link: Referenced via a URL. The content should be a CSS URL.
    * text/css (default): Injected as CSS text directly into the page.
* Content: Fill in the corresponding content based on the style type.
