# Play Guide

Without any presets, the interface is blank and you cannot interact directly. You need at least one display script to see the interaction page. This means all interactive content is customizable.

![Play](../../images/slot.png)

## Feature Descriptions

* History Paging: Jump to the corresponding interaction history.
* Output Paging: Jump and switch output. Switching output will change the context.
* Generate Image: Use the [ComfyUI](../comfyui/readme.md) image generation feature.
* Images: The [story](readme.md)'s image gallery, which can be opened as a popup window.
* Delete Reply: Delete the AI's current output.
* Delete History: Delete the input and output of the current history entry.
* Regenerate: Regenerate the AI reply.
* View Context: View the current context content and count the word count.
* Edit Current History: Edit the current history content, including input, output, and variables.
* Return to Home: Return to the story page.

* Input Box: Enter the content you want to send.
* Summary: Indicates whether this is a summary. If so, subsequent conversations will start from here and will not include previous text.
