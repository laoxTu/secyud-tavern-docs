## 什么是 Secyud Tavern

Secyud Tavern 是一个高度可定制的 AI 角色互动平台。你可以在上面搭建属于自己的 AI 互动故事，自由控制 AI 的行为、世界的规则、界面的外观，甚至让 AI 帮你画画。

## 主要功能

### 🤖 灵活的 AI 模型配置

支持接入多种 AI 大语言模型（如 OpenAI、DeepSeek 等），你可以为每个故事选择不同的模型，自由配置 API 参数。API Key 会加密存储，保障数据安全。

### 📖 世界书 — AI 的知识库

为 AI 设定世界观和背景知识。世界书会根据对话内容自动匹配相关设定注入到上下文中，让 AI 始终"记得"故事的世界规则。支持关键词匹配、时间匹配和语义匹配等多种方式。

### 🔄 正则替换 — 精确的文本控制

使用正则表达式对 AI 的输入或输出进行精准替换，可以过滤不需要的内容，或自动修正 AI 的回复。

### 🎨 自定义样式 — 改造界面外观

通过 CSS 样式表，你可以完全自定义互动界面的外观。支持内联样式和外部链接两种方式，打造你专属的视觉风格。

### 📜 自定义脚本 — 无限交互可能

通过 JavaScript 脚本，你可以在互动页面中实现各种交互效果。支持直接内嵌代码、ES 模块、外部链接和 import map 等多种方式。一切交互内容都可以由你定制。

### 🔧 宏 — 动态内容替换

在对话过程中动态计算和替换文本。比如你可以定义一个 `user` 宏，在任意文本中写入模板标记，AI 看到时会自动替换为对应的值。支持 JavaScript 表达式，灵活度极高。

### 🎬 故事管理

- **多存档**：每个故事是一个独立存档，互不影响
- **预设组合**：一个故事可以同时加载多个预设（包含世界书、正则、样式、脚本、宏），灵活组合
- **对话管理**：支持翻页历史、切换输出分支、重新生成、删除回复等完整操作
- **上下文查看**：随时查看当前发送给 AI 的完整内容

### 🖼️ AI 生图（ComfyUI 集成）

- 连接 ComfyUI，在互动中随时生成图片
- 支持从模型网站（如 CivitAI）导入模型，也可手动管理
- 支持 Diffusion Model、LoRA、VAE 等多种模型类型
- 可视化工作流配置，灵活搭建生图流程
- AI 可根据当前对话内容自动生成提示词
- 生成图片可自动保存到故事图集中

### 🔌 插件系统

支持安装外部插件扩展功能，让平台能力不断增长。

### 🌍 多语言

内置中文和英文双语言支持，在设置中即可切换。
